# Desandu Electronics - Shop Website

This is a professional landing page for Desandu Electronics, a shop in Sri Lanka specializing in Singer electronics and Tomahawk bicycles.

## Features

- **Modern Design**: Clean, professional layout with mobile-first approach
- **Product Categories**: Home Appliances, Electronics & Audio, Bicycles, Daily Essentials
- **Contact Integration**: WhatsApp and phone call buttons for easy customer contact
- **Location Info**: Complete address and opening hours
- **Trust Badges**: Authorized dealer badges for Singer and Tomahawk

## Tech Stack

- **Next.js 16** - React framework with App Router
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Beautiful icon library
- **TypeScript** - Type-safe development

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
